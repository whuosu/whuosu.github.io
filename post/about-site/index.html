# 关于本站

2020-08-22 2 min read # 生活

楔子

二〇一九年考进北京林业大学之后，我开始陷入过一段短暂的迷茫，那段时间昏天黑地，心里有种种不如意：我的学校不怎么有名气，我的专业没有前景，宿舍晾衣服的地方太小，为什么没有上床下桌这些都能成为我心情不愉快的引子。

再然后，和隔壁寝室的同学侃侃，打电子游戏，好像时间过得很快。现实中用京东购物和电子产品麻痹自己。期间也买了基本非常Professional的书本放在床头，游戏打累了就翻两下。我心里想着：过完这四年我就可以换一个学校，换一个普通的研究生身份念书，之后回到家乡和父母身边，过完平淡的一生。

二〇二〇年的春假，一放就放了九个月。转眼间我在家已经九个月了，马上就要开学了。

现在，我还是想甘于平凡。

关于搭建

网站托管于Github个人仓库，由阿里云进行域名重定向。

网站的构建和写作由Gridea支援。

由于本人粗心大意，在折腾黑苹果时没有备份Gridea的本地文件，导致数篇文章丢失。这一次一定会定期进行冷盘备份和云端备份。

关于愿景

搭建这个网站，说实话我对能不能用它来记录平时的知识点和反馈是不报有希望的。
大抵是想让它能帮助我专心写作，定期反思一下自己，记录大学四年（或者更多）的生活，心情，情感。

千尺万寻，我以绝之。

既然决定去做了，就要做完。

二〇二〇 庚子年 捌月 贰拾柒日
于长兴家中
